Try A Different Experience With Node.js JavaScript Debugging.

Powerful JavaScript debugging for Node.js to fix issues efficiently and debug code less painful. Easy, fast and stable. Try the new taste on Linux, OS X and Windows.

Cross-platform support

ironNode works best with Windows, Linux and OS X. Read supported-platforms.md for more details.

Installation

$ npm install -g iron-node

Getting started

The usage is the same as the node command. Place the keyword debugger; anywhere into your code and use iron-node instead of node. To run a debug session with the given JavaScript file type...


iron-node PATH_TO_YOUR_JAVASCRIPT_FILE [--customparm1=foo --customparm2=bar];

Debug command line apps like Grunt, Gulp, Mocha and much more
